# udl-bundled-tools

Build and publish pinned third-party macOS binaries used by UDL release bundles.

Current scope:

- `scdl` `3.0.1`

Release tags follow `scdl-v*` and publish:

- `scdl-darwin-amd64`
- `scdl-darwin-arm64`
- `SHA256SUMS`

These assets are consumed by the `adamhalama/music-library-sync` release workflow.
